## Demonic Pacts System

Replaces the Combat Masteries from previous leagues. Less linear, more flexible. Called "Demonic Pacts" fitting the Yama/demon theme. Details being revealed daily leading up to April 15.

## Echo Bosses

Echo variants for all regional bosses. Unlocked by getting Echo Orbs from standard boss kills. Enhanced rewards from Echo versions. Not all previous Echo rewards returning — some replaced with new thematic drops.

## Varlamore Content Available at Start

**Bosses:** Amoxliatl, Doom of Mokhaiotl, Gemstone Crab, Hueycoatl, Moons of Peril, Sol Heredit, Vardorvis

**Skilling:** Sunfire Runecrafting, Forestry, extensive shops

### Maxing Phases

**Phase 1 — Varlamore Only (Days 1-3):**
Prayer (Ralos' Rise), Agility (Colossal Wyrm), Thieving (Bazaar), Mining (Calcified/Stars), Runecraft (Sunfire), Construction + Fletching (Vale Totems), Hunter (Moths), Cooking, Herblore, Crafting — all buyable/trainable. Start combat at Sulphur Naguas.

**Phase 2 — First Region Unlock → Kourend + Karamja auto-unlock (Day 3-4):**
Firemaking (Wintertodt), Woodcutting (start Redwoods at 90), Smithing (Blast Furnace), Slayer (Catacombs). Fishing (Karambwans on Karamja as AFK).

**Phase 3 — Desert Unlock:**
Ancient Magicks → Ice Barrage Slayer in Catacombs (Magic + Defence). Start ToA for Tumeken's Shadow. Farming via constant runs with 1-min ticks.

**Phase 4 — Kandarin Unlock:**
Demonic Gorillas for zenytes → Tormented Bracelet → Confliction Gauntlets. Barbarian Fishing to 99 if not done. Chinchompas for Ranged 99. Lunar spells unlocked.

**Phase 5 — Mop Up + Points:**
Finish any remaining 99s. Focus on league tasks for points. Farm ToA/CoX for drops + Echo bosses for points.
