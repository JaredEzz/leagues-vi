## Pact Adjustments

| Pact | Status | Notes |
|---|---|---|
| **F8** (powered staff 1-tick) | **CORE — unchanged** | EoA at 1-tick with crystal = BIS boss DPS |
| **F7** (combat spell speed) | **Still good** | Speeds up SQ elemental casts if SQ isn't "powered" |
| **L6** (ice = water) | **DROP** | No Ancients spellbook. Ice barrage unavailable. |
| **I2** (water +20% at full HP) | **ADJUST** | Only useful if casting water spells via SQ auto-weakness. Still good for water-weak targets. |
| **I1** (air +7% per prayer) | **CONSIDER** | SQ auto-targets weakness. Air-weak targets get I1 bonus. |
| **I3** (fire burn for damage) | **CONSIDER** | Fire-weak targets. SQ auto-targets. |
| **I4** (earth reduces def) | **CONSIDER** | Earth-weak targets. Stacking debuff. |
| **FA** (magic dmg stacks) | **CORE — unchanged** | Universal damage boost |
| **C2** (water regen → healing) | **Still works** | SQ generates rune regens. Healing sustains full HP for I2 |
| **C1/C3/C4** | **Still work** | SQ uses all elements. All regen procs are active |
| **B1** (regen → magic boost) | **Still works** | Universal |

**Key change:** Drop L6 (ice = water) since no Ancients. Replace with elemental-specific pacts (I1/I3/I4) since SQ cycles through all elements based on target weakness. This is actually MORE versatile — you benefit from ALL four elemental pacts depending on what you're fighting, instead of being locked to ice.

### New pact priority

1. **F8** — EoA 1-tick (core)
2. **FA** — magic damage stacks (core)
3. **F7** — combat spell speed for SQ (core)
4. **I2** — water spells +20% at full HP (great for water-weak targets)
5. **I1** — air spells +7% per prayer (great for air-weak targets)
6. **I4** — earth spells reduce def (stacking debuff)
7. **I3** — fire spells burn damage (risk/reward)
8. **C2** — water regen healing (sustain)
9. **B1** — regen → magic boost
10. **C1/C3/C4** — regen utilities

## Dual-Style Armor: Mage AND Ranged from One Set

Crystal armor + blessing works for **both Bowfa (ranged) and Eye of Ayak (magic)**. Same armor, same blessing. You just swap weapons:

| Style | Weapon | Crystal Bonus | Use Case |
|---|---|---|---|
| **Magic (bossing)** | Eye of Ayak + DE | 1.3x acc, 1.15x dmg (powered staff) | Single-target bosses, CoX, Inferno |
| **Magic (Slayer)** | Shadowflame Quadrant + DE | 1.3x acc, 1.15x dmg (if powered) | AoE Slayer with echo |
| **Ranged (bossing)** | Bowfa | 1.3x acc, 1.15x dmg (crystal bow) | Ranged-weak bosses, Inferno rangers |

**One armor set for two combat styles.** No need to farm separate mage and ranged armor. Ancestral + Armadyl would require two separate grinds from CoX + Kree'arra. Crystal does both.

## Smithing Without Giants' Foundry — Detailed Solve

This is the biggest skill loss. Options:

### Option A: Anvil Smithing with EH Bars (Primary)

1. AFK mine ore with EH (2-tick, 2x, infinite, auto-bank)
2. Hotfoot coal-free auto-smelts into bars
3. Smith at anvil with Flow State 2-tick

| Method | Base XP/hr | At 16x + 2-tick | Notes |
|---|---|---|---|
| Rune platelegs (99 Smithing, 3 bars) | ~75 XP × ~1,200/hr | ~1.8M/hr | Need 99 first (chicken-egg problem) |
| Addy platelegs (88 Smithing, 3 bars) | ~62.5 XP × ~1,200/hr | ~1.5M/hr | More accessible |
| Mithril platelegs (68 Smithing, 3 bars) | ~50 XP × ~1,200/hr | ~1.2M/hr | Early game |

Time to 99: ~13M XP / ~1.5M/hr = **~8.7 hrs** at addy platelegs (16x + 2-tick)

### Option B: Zalcano (Prifddinas)

- Mining + Smithing XP per kill
- ~40-50k Smithing XP/hr base → ~640-800k/hr at 16x
- Also drops d-pick, crystal shards, ores, GP
- **~16-20 hrs to 99 Smithing from Zalcano alone** — too slow as primary, great as supplement

### Option C: Hybrid (Recommended)

1. **Early game (1-68):** Anvil smithing with mithril bars from EH mining. Quick at multipliers.
2. **Mid game (68-88):** Addy platelegs at anvil. ~1.5M/hr. Supplement with Zalcano for d-pick + shards.
3. **Post-88:** Rune platelegs if you have rune bars stockpiled. Or continue addy.
4. **Total: ~7-9 hrs** (vs ~3.3-6 hrs with Giants' Foundry)

**Net loss: ~3-4 hrs extra on Smithing.** This is the cost of the Tirannwn swap. Worth it for crystal BIS mage + crystal tools + Prifddinas + CG.

## Summary: Why Crystal Mage

1. **BIS mage in hours, not weeks.** Crystal + blessing from CG = 1.3x accuracy, 1.15x damage, +24 mage atk, +6% dmg. Farmed in a day by someone good at CG.
2. **EoA at 1-tick with crystal = 25 DPS.** Highest single-target mage DPS in the league. Was ~10.5 DPS with Ancestral.
3. **One armor set for mage AND ranged.** Crystal + Bowfa for ranged, crystal + EoA for mage. No separate armor grinds.
4. **Crystal tools upgrade entire gathering engine.** Crystal pick/axe/harpoon > dragon for EH + Flow State mining/WC/fishing.
5. **Prifddinas = best Agility + Thieving.** Saves hours on two skills.
6. **Don't need Ancients.** SQ auto-elemental + echo + doubled weakness out-DPS ice barrage on single targets. All four elemental pacts (I1-I4) activate based on target weakness.
7. **CG replaces ToA as endgame solo content.** You're good at CG. It's more skill-expression per hour than ToA.
8. **Only costs ~3-4 hrs on Smithing.** Anvil smithing + Zalcano replace Giants' Foundry.
