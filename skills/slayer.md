# Slayer — Skill Guide

**Target:** 99 | **Regions:** Kourend + Desert (barrage Slayer) | **Key Relic: Culling Spree (Tier 6)**

## Time Estimates

| Scenario | Est. Time to 99 |
|----------|-----------------|
| Base (16x, no relic) | ~8 hrs |
| With Abundance | **~8 hrs** (combat skill — no +10 boost) |
| With Culling Spree (T6) | **~5-6 hrs** (task selection + superior chains + free helm = wear Ancestral) |

**Abundance saves ~5 min**. Combat skill — no +10 boost. Culling Spree is the real time saver here (choose barrage tasks, superior chaining). Also trains Magic + Defence simultaneously via defensive barrage.

## Culling Spree Relic (Tier 6)

This changes everything for Slayer:

| Perk | Effect |
|------|--------|
| **Choose your task** | Pick from 3 task options — always pick barrageable/best XP tasks |
| **Custom kill count** | Set how many kills per task — short tasks for fast points, long tasks for XP |
| **Superior chaining** | Superiors have 50% chance to spawn ANOTHER superior on death (chain kills!) |
| **Elite clue drops** | Superiors always drop 1-3 elite clue scrolls (stackable in leagues) |
| **Free Slayer helm effects** | All Slayer helm bonuses without wearing one — wear Ancestral Hat instead! |
| **Free shop perks** | All Slayer reward unlocks are free (broader fletching, broader bolts, etc.) |
| **Utility items** | Rune pouch, Herb sack, Looting bag from Slayer store |

### Why This Is Broken

- **Free Slayer helm = wear Ancestral Hat** for +8 magic atk and +3% magic dmg while still getting the Slayer helm damage/accuracy bonus
- **Choose your task** = always pick Dust Devils or Nechryaels for barrage, or Smoke Devils at 93
- **Superior chaining** at 1/50 base spawn rate + 50% chain = insane bonus XP and elite clues
- **Custom kill count** = set short tasks to rush Slayer points, or long tasks when you get a good barrage assignment

## Best Strategy to 99

### Pre-Culling Spree (Before Tier 6)

Train Slayer normally through combat. Sulphur Naguas (Varlamore) for early melee Slayer XP. Once Kourend + Desert are unlocked, start barrage tasks in Catacombs.

### With Culling Spree (Tier 6+)

#### Priority Tasks (Always Pick These)

| Task | Slayer Lvl | Method | Why |
|------|-----------|--------|-----|
| **Dust Devils** | 65 | Ice Burst/Barrage (Catacombs) | Multi-combat, stack and barrage. Best early barrage task |
| **Nechryaels** | 80 | Ice Barrage (Catacombs) | Best barrage task overall. Big stacks, great drops |
| **Smoke Devils** | 93 | Ice Barrage (Smoke Devil Dungeon) | Thermonuclear boss drops Occult Necklace. Barrageable |

#### How to Barrage Tasks

1. Enter Catacombs of Kourend (multi-combat)
2. Equip Kodai Wand (autocasts Ancients, unlimited water runes)
3. Wear full Ancestral (no Slayer helm needed — Culling Spree gives the bonus for free!)
4. Stack monsters → Ice Barrage → repeat
5. Superiors spawn at 1/50, then 50% chance to chain another superior
6. Each superior drops 1-3 elite clues (stackable)

### Leveling Plan

1. **1-50:** Melee Slayer — Sulphur Naguas (Varlamore), any Slayer master available
2. **50-65:** Unlock Kourend, start Catacombs tasks. Melee or early burst spells
3. **65-80:** Unlock Desert → Ancient Magicks. **Ice Burst Dust Devils** in Catacombs. This is where Slayer gets fast
4. **80-93:** **Ice Barrage Nechryaels** in Catacombs. With Culling Spree, always choose Nechs
5. **93-99:** **Smoke Devils** (Kandarin). Farm Thermonuclear for Occult Necklace. Alternate with Nechs

### Superior Chaining Math

- Base superior rate: 1/50
- Culling Spree: 50% chance to chain another on death
- Expected superiors per task (100 kills): ~2 base + ~1 chain = ~3 superiors
- Each drops 1-3 elite clues
- **~3-9 elite clues per task** — insane for league points from clue completions
