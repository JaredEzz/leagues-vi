# Fletching — Skill Guide

**Target:** 99 | **Regions:** Varlamore, Kandarin

## Time Estimates

| Scenario | Est. Time to 99 |
|----------|-----------------|
| Base (16x, no relic) | ~4.3 hrs |
| With Abundance only | **~4 hrs** (+10 boost = higher recipes earlier) |
| With Abundance + Hotfoot | ~4 hrs (Hotfoot doesn't affect Fletching) |
| With Abundance + Woodsman | **~0.5-1 hr** (all items process simultaneously, stackable 10x per action) |

**Abundance saves ~20 min** (+10 boost = earlier recipe access). +2/drop = +1% (negligible).

## Methods

| Level Range | Method | Region | Notes |
|-------------|--------|--------|-------|
| 1-33 | Arrow shafts / basic bows | Any | Quick start |
| 33-66 | Vale Totems (Auburnvale) | Varlamore | Trains Construction + Fletching together |
| 66-99 | Yew/Magic longbows | Kandarin / Varlamore | Spin flax near Seers (Ka) for bowstrings. Cut logs from WC training |

## Materials

| Material | Source | Region |
|----------|--------|--------|
| Logs | Forestry (Varlamore), Redwoods (Kourend) | V / Ko |
| Bowstrings | Spin flax (Seers' Village) | Kandarin |
| Feathers | Fishing shops / chicken drops | Various |

## Notes

- Vale Totems are great dual-skill training (Fletching + Construction)
- Flax field near Seers' Village (Kandarin) for bowstrings
- Abundance +10 boost helps with recipe access
